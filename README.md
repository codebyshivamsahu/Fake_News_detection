This is a **Machine Learning academic project** developed by Computer Science Engineering students. The system uses Natural Language Processing (NLP) and a trained classification model to analyze news headlines or article text and predict whether the news is **REAL** or **FAKE**.

The web application is built with **Flask** and styled using **Bootstrap** and **Tailwind CSS** for a clean, responsive user experience.
