About Us

This Fake News Detection project is a machine learning-based web application developed as part of our academic curriculum. The goal is to combat the spread of misinformation by providing a reliable tool that analyzes news articles and classifies them as real or fake using advanced natural language processing techniques.

Objective

To build an accurate and efficient system for detecting fake news using machine learning models.

Technology Used

Python, Scikit-learn, NLTK, Flask/FastAPI, Bootstrap & Tailwind CSS
